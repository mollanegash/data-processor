# Data Processor Application

## Overview

The **Data Processor Application** is a full-stack application designed for processing and analyzing data based on NAICS codes from CSV files. The application provides functionality for filtering, transformation, and reporting. It uses Java (Spring Boot) for the backend, React for the frontend, and MySQL for database management.

## Features

- **Backend (Java Spring Boot)**:
  - RESTful APIs for data processing
  - Integration with MySQL database
  - CSV file handling and data processing

- **Frontend (React)**:
  - File upload functionality
  - Data visualization and reporting
  - User interface for filtering and sorting data

- **Database (MySQL)**:
  - Schema for storing processed data
  - Efficient querying and reporting

## Technologies

- **Java**: Spring Boot
- **React**: Frontend framework
- **MySQL**: Database
- **Maven**: Build tool

## Prerequisites

- **Java**: JDK 22 or later
- **Node.js**: For running React development server
- **MySQL**: For database management
- **Maven**: For building the Java application
